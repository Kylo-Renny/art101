Pseudocoding an Everyday Task

  1. Find another car
  2. Connect Jumper Cables
    • Attach red to red of your battery, then red to red of the other car
    • Attach black to black of the other car, then black to metal part of dead battery
  3. Start the other Cars Engine
  4. Start the Dead Car
    • Accelerate a bit on other car if jumpstart does not work after few tries
  5. Take Cables out in Reverse Order

Pseudocoding the Dinosaur Game

  1. Generate a Desert Landscape
    • Include Cacti in varying groups of 1 to 4
    • Include Birds
  2. Create Dinosaur Character
    • Include Animation for Running, Jumping, Ducking, and Death
  3. Implement Controls
    • Space Button Starts Game and used for Jumping
    • Down Arrow for Ducking
  4. Create Variables
    • Implement randomizer for Cacti and Bird obstacles
    • Score begins at 0 and increases by distance Dinosaur goes
